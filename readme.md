# VBB - Tự động điền form từ mã Json

*Develop 06-11-2020.*  
Tiện ích Chrome - điền form từ mã Json.  
Tải tiện ích tại:  
<https://www.nguyenvubang.com>  

Điền form thủ công lặp đi lặp lại, tốn thời gian, dễ phát sinh sai sót.
Lưu dữ liệu thành mã Json and sử dụng nhiều lần để tự động điền form.

## Ví dụ

For now the extension has permission to modify form data only on this websites:  
`https://www.nguyenvubang.com/`  
`http://dvc.mt.gov.vn/*`  

Form demo: Mở file Json  
<https://www.nguyenvubang.com/>  
copy vào clipboard.  
Sau đó mở trang  
<https://www.nguyenvubang.com/>  
chạy tiện ích và paste dữ liệu từ clipboard.  
Click `Điền Form`.  
Done.  

## Cài đặt

Bạn có thể cài tiện ích"  
Tại `<chrome://extensions/>` Bật `Developer mode` và tải `Load unpacked`.
